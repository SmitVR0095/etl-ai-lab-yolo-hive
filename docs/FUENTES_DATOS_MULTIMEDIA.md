# Fuentes de datos multimedia

El repositorio no incluye las imágenes ni videos originales para evitar subir archivos pesados.

Los archivos multimedia utilizados en las pruebas del proyecto se entregan por separado en un archivo comprimido.

## Fuentes utilizadas

### Imágenes

- Fuente: Pexels
- Búsqueda utilizada: datos
- URL: https://www.pexels.com/es-es/buscar/datos/

### Videos

- Fuente: Pixabay
- Búsqueda utilizada: personas
- URL: https://pixabay.com/es/videos/search/personas/
